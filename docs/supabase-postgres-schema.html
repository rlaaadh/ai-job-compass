Supabase Postgres Schema

Supabase/Postgres 기준 DB 스키마 문서

이 문서는 현재 이직각 프로젝트가 Supabase(Postgres)로 넘어갈 때 기준이 되는 핵심 테이블과 컬럼, 인덱스, 관계를 정리한 설계 문서입니다. 목표는 회사 검색을 빠르게 만들고, 이후 건강도 계산과 비교 분석까지 자연스럽게 연결하는 것입니다.

핵심 테이블

companies

검색 결과 목록과 상세 분석의 기준이 되는 메인 테이블

company_monthly_stats

월별 직원 수, 입사자, 퇴사자 등 건강도 계산용 테이블

company_search_aliases

별칭과 정규화 검색을 담당하는 검색 보조 테이블

추천 검색 쿼리 방향

1차 검색

정규화된 이름의 앞부분(prefix) 일치 우선

WHERE normalized_name LIKE :query || '%'
ORDER BY search_rank DESC, employee_count DESC
LIMIT 10

2차 보조 검색

별칭/부분일치 보조

JOIN company_search_aliases a ON a.seq = c.seq
WHERE a.normalized_alias_text LIKE :query || '%'
ORDER BY c.search_rank DESC, c.employee_count DESC
LIMIT 10
현재 코드에 반영한 방향
