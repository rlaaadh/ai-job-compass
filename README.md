# ai-job-compass "다른 직장은 어떨까 궁금하다면? 이직각 ."
**이직각** : 이 프로젝트는 국민연금 공공데이터를 기반으로 기업 건강도와 이직 추천도를 제공하는 서비스입니다.
<br/>
<br/>

# 한 줄 소개
국민연금 가입자 변동 공공데이터를 AI가 분석하여 기업의 성장성과 안정성을 평가하고, 
현재 직장 대비 이직 추천도와 예상 리스크를 제공하는 AI 기반 이직 의사결정 서비스이다.

실제로 직장인이 궁금한 것은 단순한 기업 정보가 아니라, 
"이직 시 어떤 리스크가 존재하는가?", "현재 회사보다 더 좋은 선택인가?", "이 회사는 실제로 성장하고 있는가?" 등과 같은 사항들이다.
**이 프로젝트는 기업의 실제 인원 변화를 분석하고, AI가 이를 해석하여 기업 건강도와 이직 추천도를 제공한다.**

<br/>
<br/>

# 핵심 기능
## 1) 기업 검색
회사명을 검색하면 국민연금 데이터를 기반으로 기업 정보를 조회한다.

**제공 정보**
<ul>
  <li>현재 직원 수</li>
  <li>신규 입사자 수</li>
  <li>퇴사자 수</li>
  <li>최근 1년 직원 수 변화</li>
  <li>직원 증가율</li>
  <li>조직 규모 변화 추이</li>
</ul>

## 2) AI 기업 건강도 분석
국민연금 가입자 변동 데이터를 기반으로 기업의 조직 상태를 분석한다.

**분석 항목**
<ul>
  <li>성장성</li>
  <li>안정성</li>
  <li>기업 규모</li>
</ul>

**현재 점수 체계 (100점 만점)**
<ul>
  <li>성장성 40점 : 최근 1년 직원 수 증감률과 최근 직원 수 흐름</li>
  <li>안정성 35점 : 이직률, 월별 변동 안정성, 최근 감소 방어력</li>
  <li>기업 규모 25점 : 현재 국민연금 가입 직원 수 기준</li>
  <li>리스크 패널티 : 최근 3개월 연속 감소, 급격한 인원 감소 시 감점</li>
</ul>


**예시**
<ul>
  <li>기업 건강도 : 82점</li>
  <li>성장성 : 높음</li>
  <li>안정성 : 양호</li>
  <li>기업 규모 : 중견 기업 수준</li>
</ul>

## 3) AI 기업 분석 리포트
AI가 수치를 사람이 이해하기 쉬운 자연어로 해석한다.

예시
"최근 12개월 동안 직원 수가 15% 증가했으며 퇴사자 대비 입사자 비율이 높아요. 현재 조직은 확장 단계에 있으며 안정성 또한 양호한 수준입니다."

## 4) 현재 직장 대비 기업 비교
현재 재직 중인 회사와 관심 기업을 비교한다.

**비교 항목**
<ul>
  <li>직원 수</li>
  <li>조직 규모</li>
  <li>직원 증가율</li>
  <li>최근 직원 수 감소 여부</li>
  <li>월별 변동성</li>
</ul>

## 5) AI 이직 추천도 ⭐️⭐️⭐️ (서비스의 핵심 차별화 기능)
현재 회사와 관심 기업을 비교하여 AI가 이직 추천도를 계산한다.

**평가 요소**
<ul>
  <li>조직 성장성</li>
  <li>조직 안정성</li>
  <li>조직 규모</li>
  <li>예상 리스크</li>
</ul>


**예시**
<ul>
  <li>이직 추천도 : 84점</li>
  <li>성장성 : ★★★★★</li>
  <li>안정성 : ★★★★☆</li>
  <li>기업 규모 : 중견 기업 수준</li>
  <li>예상 리스크 : 제조업 특성상 IT 조직 규모 제한 가능 / 조직 문화 적응 필요</li>
</ul>


종합 의견
"현재 회사 대비 성장성과 연봉 측면에서 긍정적인 선택으로 판단됩니다."
